Description

The semester project for CEN4090L (Software Engineering Capstone).

  • Full-stack python web application built on Gunicorn Flask.
  • Hosted by an NGINX web server served on an Ubuntu 20.04 instance.
  • News tab is populated through interacting with HackerNewsAPI.
  • User management and authorization is handled through Auth0.
